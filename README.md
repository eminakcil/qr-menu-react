# Başlarken
Kategorilerin ve ürünlerin yönetilebildiği menü uygulaması.

## Kurulum

Paketleri kurmak için bu komutu çalıştırın.
```bash
npm install
```

Geliştirme sunucusunu başlatmak için bu komutu çalıştırın.
```bash
npm run dev
```
